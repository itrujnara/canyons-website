# Canyons Website

<img src="https://www.museothyssen.org/sites/default/files/imagen/2023-10/SHEELER_Ca%C3%B1ones_757_%281973.6%29_%23GEXTENSA.jpg" height="200">

This is a simple vanilla HTML webpage, showcasing a design system based on the painting "Canyons" by Charles Sheeler. The system is based on 6 colors, 2 fonts, and grid-based 3D shapes. The page is highly responsive and should render correctly on most devices.

## Features

- Custom color palette and font selection
- Parallax scrolling
- True 3D rendering
- 4-level responsive design

## Technology stack

| Feature   | Framework     |
| --------- | ------------- |
| Structure | Vanilla HTML5 |
| Style     | Tailwind CSS  |
| Scripts   | Typescript    |
| Packaging | Vite          |

## AI disclosure

- GitHub Copilot next edit prediction was used throughout
- Gemini was used to troubleshoot several CSS issues
- `scroll.ts` was mostly generated by Gemini
